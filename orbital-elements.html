Explanation of Orbital Elements

Semi-major axis (a)
The semi-major axis is half the longest diameter of an elliptical orbit. It determines the size of the orbit and is directly related to the orbital period.
Eccentricity (e)
Eccentricity describes the shape of the orbit. It ranges from 0 (circular orbit) to 1 (parabolic trajectory). Values between 0 and 1 represent elliptical orbits.
Inclination (i)
The inclination is the angle between the orbital plane and the reference plane (usually the Earth's equatorial plane). It ranges from 0° to 180°.
Right Ascension of Ascending Node (Ω)
This is the angle from the reference direction to the ascending node (where the orbit crosses the reference plane going north). It ranges from 0° to 360°.
Argument of Periapsis (ω)
The argument of periapsis is the angle from the ascending node to the periapsis (point of closest approach) measured in the orbital plane. It ranges from 0° to 360°.
True Anomaly (ν)
True anomaly is the angle from the periapsis to the current position of the orbiting body, measured in the direction of motion. It ranges from 0° to 360°.
